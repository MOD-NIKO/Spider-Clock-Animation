# MOD NIKO Animated Clock

A visually stunning animated clock built with **HTML**, **CSS**, and **GSAP (GreenSock Animation Platform)**. This project demonstrates complex SVG animations, including rotating cogs, moving clock hands, and custom shadows to create a dynamic and interactive clock.

## Features

- Fully animated SVG clock with **hour, minute, and second hands**.
- Rotating cogs with clockwise and counterclockwise animation.
- Realistic shadow and blur effects using SVG filters.
- Responsive design with proper **viewport scaling**.
- GSAP-powered animations for smooth and performant motion.
- Modular and reusable SVG components.

## Technologies Used

- **HTML5** – Structure of the project
- **CSS3** – Styling and positioning of elements
- **SVG** – Vector graphics for clock and gears
- **GSAP (GreenSock)** – Advanced animation framework
  - `MotionPathPlugin`
  - `ScrollTrigger`
